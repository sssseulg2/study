react express chat 
=======
ionic을 활용해 프론트 구현을 통해 angular와 typescript 실력 향상
(서버는 이전 스터디에서 사용했던 express.js 사용)

<br>

### client 📁
```
cd client
npm install
ionic serve
```

### server 📁
```
cd server
npm install
npm start (or nodemon server.js)
```
